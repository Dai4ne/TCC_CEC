# TCC_CEC
CEC (Controle de Equipamentos do Candelária)

O projeto aborda a necessidade de modernização na gestão de equipamentos da Escola Estadual Joaquim de Moura Candelária, onde o controle era feito manualmente. Diante desse cenário, o CEC tem um sistema informatizado para gerenciar os equipamentos, permitindo o registro centralizado de entradas, saídas, devoluções, manutenções e envio de notificações. A solução proposta justifica-se por sua capacidade de reduzir erros, ampliar a segurança dos aparelhos eletrônicos e facilitar a tomada de decisões, tornando o gerenciamento de equipamentos mais rápido, organizado e confiável.
